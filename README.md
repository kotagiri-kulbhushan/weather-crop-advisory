# Weather and Crop Advisory System

This project is a Python-based weather and crop advisory system developed using Jupyter Notebook.  
It analyzes weather data and provides basic crop-related insights useful for farmers and students.

## Features
- Weather data analysis
- Crop advisory insights
- Clean and beginner-friendly implementation

## Tech Stack
- Python
- Jupyter Notebook
- Pandas
- Requests
- Matplotlib

## How to Use
1. Clone or download the repository
2. Open the notebook in Jupyter
3. Run all cells step by step

## Use Case
- Educational projects
- Weather analysis
- Agriculture-related mini projects
